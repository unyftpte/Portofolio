SimpleFOC BLDC 3 Phase Research

Research on the 3-phase BLDC controller at the Garuda UNY Student Activity Unit (UKM) aims to develop a controller using the SimpleFOC algorithm with an STM32 Nucleo microcontroller. Preliminary results show that the controller successfully operates on a 3-phase BLDC with a voltage range of 36-48V and a maximum current of 10A. This project is ongoing and has not yet been used in any official competition.

Analysis Performance Motor by: Siemens Motorsolve

This project involves the use of Motorsolve software to design and analyze brushless DC (BLDC) motors. The main focus is to optimize the stator windings, coil configuration, and thermal management to improve motor performance. Key parameters such as winding resistance, coil layout, and material selection are defined to ensure efficient operation and good thermal management. This software also supports motor performance visualization and testing through simulation.

Design and Simulation Stator and Winding Analysis Electric Motor Fundamentals Thermal Management
